# Simple Interest Calculator Script

This Bash script calculates simple interest given the principal amount, annual rate of interest, and time period in years. Please note that this script is for demonstration purposes only and should not be used in production environments.

## Usage

### Input
- `p`:  Principal amount
- `t`:  Time period in years
- `r`:  Annual rate of interest

### Output
- `simple_interest`: Calculated simple interest using the formula simple_interest = p * t * r / 100

## How to Use
1. Clone or download the script to your local machine.
2. Open a terminal.
3. Navigate to the directory where the script is located.
4. Run the script by executing the following command:
   ```bash
   ./simple_interest_calculator.sh
   ```
5. Follow the on-screen prompts to enter the required values:
   - Enter the principal amount `p`.
   - Enter the annual rate of interest `r`.
   - Enter the time period in years `t`.
6. The script will calculate and display the simple interest.

### Sample Usage

```yaml
$ ./simple_interest_calculator.sh
Enter the principal:
1000
Enter rate of interest per year:
5
Enter time period in years:
2
The simple interest is:
100
```
